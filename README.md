A comprehensive, self-paced SQL course designed to take you from beginner to advanced SQL mastery through progressive lessons and hands-on exercises.

## Course Overview

This course is structured into 8 progressive levels, each building upon previous concepts. Each level contains:
- **Lessons**: Clear explanations with examples and best practices
- **Exercises**: Practical problems with varying difficulty levels
- **Your Solutions**: You'll create your own solutions to reinforce learning

## Course Structure

### Level 1: SQL Fundamentals
- Introduction to SQL and Databases
- SELECT Basics
- Filtering and Sorting
- *5-7 exercises*

### Level 2: Data Manipulation
- INSERT, UPDATE, DELETE
- Data Types and Constraints
- *4-5 exercises*

### Level 3: Aggregations and Grouping
- Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
- GROUP BY and HAVING
- *6-8 exercises*

### Level 4: Joins
- INNER JOIN
- LEFT/RIGHT/FULL OUTER JOIN
- Self Joins and Multiple Joins
- *8-10 exercises*

### Level 5: Subqueries
- Scalar and Column Subqueries
- EXISTS and IN with Subqueries
- *6-8 exercises*

### Level 6: Advanced Queries
- Common Table Expressions (CTEs)
- Window Functions
- *7-9 exercises*

### Level 7: Data Modeling and Design
- Normalization (1NF, 2NF, 3NF)
- Indexes and Performance
- *3-4 exercises*

### Level 8: Advanced Topics
- Stored Procedures and Functions
- Views and Triggers
- Transactions and Concurrency
- *4-5 exercises*

## How to Use This Course

Follow this step-by-step learning flow for each level:

### Step 1: Set Up Your Database
1. Navigate to the `database/` folder
2. Follow the setup instructions in `database/README.md`
3. Run the `setup.sql` script to create the practice database

### Step 2: Read the Lessons
1. Start with the first lesson in the level (e.g., `lesson-01-introduction.md`)
2. Read through all lessons in the level sequentially
3. Type out and run the example queries yourself
4. Experiment with modifying the examples to deepen understanding

### Step 3: Attempt the Exercises
1. Open the exercises file (e.g., `exercises-01.md`)
2. **Write your own SQL solutions** for each exercise
3. Test your solutions by running them against the database
4. Verify your results match the expected output format

### Step 4: Debug and Iterate
1. If a query doesn't work, read the error message carefully
2. Review the relevant lesson sections
3. Check the hints in the exercises file (only if you're stuck)
4. Refine your solution until it works correctly

### Step 5: Move Forward
1. Once you've completed all exercises in a level, move to the next level
2. Don't skip exercises - they're essential for building skills
3. Revisit previous lessons if you need a refresher

## Learning Tips

- **Start simple**: Begin with Level 1 and progress sequentially - each level builds on previous concepts
- **Practice actively**: Don't just read - write and run every query yourself
- **Use hints wisely**: Only check hints when you're genuinely stuck, not as a first resort
- **Test incrementally**: Build queries step by step, testing as you go
- **Review when needed**: Return to previous lessons if concepts need reinforcement
- **Build projects**: Apply learned concepts to real-world scenarios outside the exercises
- **Be patient**: Mastery comes through practice - don't rush through exercises
